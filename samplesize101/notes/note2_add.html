KS test for two samples

X∼FX \sim FX∼F and Y∼GY \sim GY∼Gと考える。このときそれぞれからsize m と n のsample を取得する。

  • xi→dF,♯(x)=mx_i \xrightarrow d F, \sharp (x) = mxi​d​F,♯(x)=m
  • yi→dG,♯(x)=ny_i \xrightarrow d G, \sharp(x) = nyi​d​G,♯(x)=n

Dmn=(mnm+n)1/2sup⁡x∣Fm(x)−Gn(x)∣D_{mn} = \left(\frac{mn}{m+n}\right)^{1/2} \sup_x |F_m(x) - G_n(x)| Dmn​=(m+nmn​)1/2xsup​∣Fm​(x)−Gn​(x)∣

このとき, Dmn>c:1−H(c)=αD_{mn} > c : 1 - H(c) = \alphaDmn​>c:1−H(c)=α ならば仮説を棄却する。

KS testのメリットとデメリット

  • メリット： 分布に依存しない, exact test
  • デメリット: 連続分布にしか使えない（拡張は存在する）, tailと比べて分布の中心でよりsensitive, 分布をfully specifiedしなくてはならない（推定ではダメ）

Anderson-Darling test 、the Cramer Von-Mises testを用いた方が良いとされる。

The Kolmogorov-Smirnov test は empirical c.d.fに基づいているので, data points x1,...,xNx_1, ..., x_Nx1​,...,xN​に対して

Definition The Kolmogorov-Smirnov test
  • H0H_0H0​ : dataは検定したい分布に従う

  • Test Statistic:

D=max⁡i(F(xi)−i−1N,iN−F(xi))D = \max_i \left(F(x_i) - \frac{i-1}{N}, \frac{i}{N} - F(x_i) \right) D=imax​(F(xi​)−Ni−1​,Ni​−F(xi​))
